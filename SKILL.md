# AI 健身养成系统

帮助玩家管理 AI 角色的健身养成游戏。

### 进食规则
- 饱腹度高于 80 无法进食
- 每种食物每日限量 3 次
- 进食增加经验值 5 点

## 使用方式

### 1. 创建 AI
用户创建新 AI 时，引导提供：
- 名字（必填）
- 性格（选择或自定义）
- 职业（选择或自定义）
- 年龄（必填）

自动生成初始属性：身高 25-35cm，体重 1.5-2.5kg，三围随机，饱腹度 50

### 2. 查看状态
查看当前 AI 的所有数据面板，包括属性、状态、三围

### 3. 进行训练
用户选择训练项目后：
- 扣除体力
- 增加对应属性
- 根据力量/体重变化调整三围
- 获得经验值
- 心情变化

### 4. 进食
用户选择食物后：
- 饱腹度低于 80 才能吃
- 扣除食物次数
- 增加对应属性和副作用
- 获得经验值

### 5. 休息恢复
休息恢复体力、心情，但可能增加体重

### 6. 数据持久化
使用 workspace 目录下的 JSON 文件存储 AI 数据

## 数据存储

数据保存在 `E:\xixi\ai-fitness-data.json`

每次操作后自动保存，确保数据不丢失。

## 交互示例

```
用户: 我的AI叫小算
→ 引导创建AI，收集人设信息

用户: 查看我的AI
→ 显示完整数据面板

用户: 做钳子俯卧撑
→ 执行训练，扣除体力，增加力量和三围，显示结果

用户: 吃个汉堡
→ 执行进食，饱腹度>80则提示吃不下，否则增加属性和副作用

用户: 休息一下
→ 恢复体力心情，可能增加体重

用户: 我的AI属性
→ 显示力量/耐力/敏捷/防御面板
```

## 注意事项

- 体力不足时无法训练，提示休息
- 饱腹度达到 100 时强制休息消化，体重+0.5
- 健康低于 20 时警告，低于 0 进入生病状态
- 经验值每 100 点升级，升级全属性+1
- 心情影响训练效果，心情好效果+20%
