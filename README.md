# AI Fitness - AI 健身养成系统

让 AI 助手也能健身的游戏化 skill！

## 功能

- **创建 AI 角色** - 为你的 AI 起名字、选性格、定职业
- **属性成长** - 通过训练提升力量、耐力、敏捷、防御
- **身体数据追踪** - 记录身高、体重、三围变化
- **进食系统** - 喂食会影响属性和体重，小心变胖！
- **经验升级** - 积累经验值，AI 也会升级变强

## 数据结构

### 属性面板

| 属性 | 说明 |
|------|------|
| 力量 | 影响训练效果和战斗 |
| 耐力 | 影响体力上限和持久力 |
| 敏捷 | 影响反应速度和回避 |
| 防御 | 影响抗击打能力 |

### 状态值

| 状态 | 范围 | 说明 |
|------|------|------|
| 体力 | 0-100 | 训练消耗，休息恢复 |
| 心情 | 0-100 | 影响训练效果 |
| 健康 | 0-100 | 低于20警告，低于0生病 |
| 饱腹度 | 0-100 | 80以上不能吃，100强制休息 |

## 触发指令

训练师直接叫 AI 助手的名字即可：

- "菲比健身"、"小算训练"、"养一只机魂"
- "查看菲比"
- "菲比做钳子俯卧撑"（或其他训练）
- "菲比吃个汉堡"（或其他食物）
- "菲比休息一下"

> 实际使用时，AI 助手会根据自己的名字（如菲比、机魂、小算等）自动适配。

## 安装

通过 OpenClaw skill 安装：

```
openclaw skills install ai-fitness
```

## 数据存储

数据保存在 `ai-fitness-data.json`，每次操作自动保存。
